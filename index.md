## Cách thêm bài học mới

1. Copy thư mục `course/00-template/` thành `course/NN-ten-bai-hoc/`
2. Cập nhật bảng trên và mục "Danh sách bài học" trong `CLAUDE.md`
3. Điền nguồn vào `sources/`, câu hỏi nháp vào `questions.md`
4. Mở Claude Code tại thư mục gốc, yêu cầu đọc `questions.md` + nguồn liên quan,
   rồi viết note hoàn chỉnh vào `notes.md`
